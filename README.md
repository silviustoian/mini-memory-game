# 🧠 Memory Game – React + Tailwind

A simple memory matching game built with **React**, **Vite**, and **Tailwind CSS**.

You start with 6 images that are duplicated and shuffled into 12 cards. Click to flip, match pairs, and win!

---

## 🧪 Tech Stack

- ⚛️ React
- ⚡️ Vite
- 🎨 Tailwind CSS
